# Start Guide to Init project 

## Start back end

FIRST YOU NEED A BE IN BACK DIRECTORY 

Create a venv using 

```python -m venv _env```

active a venv 

```.\_env\Scripts\activate```

 install requirements using 

 ```pip install -r requirements.txt```

 run project using 

 ```python manage.py runserver```
